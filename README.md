# perl-pond
A basic game to share other's positive points, without disclosing your name (opposite game of fish pond)


## Steps to up and running

- clone project
```
git clone https://github.com/AdiechaHK/perl-pond.git
```

- install dependancies
```
npm install
```

- run project
```
node app.js
```
